# DANN_pytorch-
Implementation of the paper Unsupervised Domain Adaptation by Backpropagation  

# Environment
Pytorch 0.4.1  
Python 3.6  

# How to run
Download mnist_m dataset from https://github.com/fungtion/DANN & put it in dir "data"  
Change data root in params.py  
Run main.py
